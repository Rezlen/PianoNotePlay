CSS Cheat Sheet Borders

Tag Name Description
border-style Border property specifies what kind of border to display.
p.dotted {border-style: dotted;} Dotted Defines a dotted border
p.dashed {border-style: dashed;} Dashed Defines a dashed border
p.solid {border-style: solid;} Solid Defines a solid border
p.double {border-style:double;} Double Defines a double border
p.groove {border-style:groove;} Groove Defines a 3D grooved border. The effect depends on the border-color value
p.ridge {border-style: ridge;} Ridge Defines a 3D ridged border. The effect depends on the border-color value
p.inset {border-style: inset;} Inset Defines a 3D inset border. The effect depends on the border-color value
p.outset {border-style: outset;} Outset Defines a 3D outset border. The effect depends on the border-color value
p.none {border-style: none;} None Defines no border
p.hidden {border-style: hidden;} Hidden Defines a hidden border
